# StudyPilot — AI Study Companion

## Overview

StudyPilot is an AI-powered study companion that organizes course content into a personalized exam plan, flashcards, practice quizzes, compact cheat sheets, and a chat interface based on your uploaded course materials.

### Features

- **Course Materials Manager**  
  Upload and manage lecture slides, notes, PDFs, and other course files once per course.

- **AI Study Schedule Planner**  
  Generate a day-by-day exam study plan that estimates time per topic based on the length and complexity of your materials.

- **AI Flashcard Generator**  
  Create term/definition style flashcard sets from text and visuals in your course files, with options to rename sets.

- **AI Quiz Generator**  
  Build multiple-choice and true/false quizzes at different difficulty levels using uploaded lecture and textbook content.

- **AI Cheat Sheet Generator**  
  Compress key formulas, definitions, and concepts into a 3×5 notecard, single page, or two-sided cheat sheet, downloadable into a PDF.

- **Course Q&A Chat**  
  Ask questions in the chat about your course content and get answers based on uploaded course materials.

## Tech Stack 

- **Language:** Python  
- **Frontend + Backend:** Streamlit (web app)  
- **AI Integration:** LLM APIs (OpenAI GPT-4o)  
- **Other Tools:** PDF and slide parsers, vector store for retrieval, SQLite or simple file-based storage

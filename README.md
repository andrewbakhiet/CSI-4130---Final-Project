# StudyPilot — AI Study Companion

## Overview

StudyPilot is an AI-powered study companion that organizes course content into a personalized exam plan, interactive flashcards, practice quizzes, compact cheat sheets, and a chat interface based on your uploaded course materials.

### Planned Features

- **Course Materials Manager**  
  Upload and manage lecture slides, notes, PDFs, and other course files once per course.

- **AI Study Schedule Planner**  
  Generate a day-by-day exam study plan that estimates time per topic based on the length and complexity of your materials.

- **AI Flashcard Generator**  
  Create term/definition style flashcard sets from text and visuals in your course files, with options to star and rename sets.

- **AI Quiz Generator**  
  Build multiple-choice and true/false quizzes at different difficulty levels using your uploaded lecture and textbook content.

- **AI Cheat Sheet Generator**  
  Compress key formulas, definitions, and concepts into a 3×5 notecard, single page, or two-sided cheat sheet.

- **Course Q&A Chat**  
  Ask questions about your course and get answers grounded in your uploaded notes and slides.

## Tech Stack (Planned)

- **Language:** Python  
- **Frontend + Backend:** Streamlit (web app)  
- **AI Integration:** LLM APIs (e.g., OpenAI GPT-4o for text + images, embeddings)  
- **Other Tools:** PDF and slide parsers, vector store for retrieval, SQLite or simple file-based storage
